Plugin Documentation

This report describes goals, parameters details, requirements and sample usage of this plugin.

Goals

Goals available for this plugin:

Goal Description
zonky:help Display help information on zonky-maven-plugin.
Call mvn zonky:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
zonky:start Goal which starts an embedded postgres database.
zonky:stop Goal which stops the embedded postgres database.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.9.5
JDK 1.8

System Requirements History

The following specifies the minimum requirements to run this Maven plugin for historical versions:

Plugin Version Maven JDK
from 0.1 to 0.6 3.9.5 8

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>de.funfried.maven.plugins</groupId>
          <artifactId>zonky-maven-plugin</artifactId>
          <version>0.7-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>de.funfried.maven.plugins</groupId>
        <artifactId>zonky-maven-plugin</artifactId>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
